# `helm/` — Helm Charts

> **Kubernetes Helm charts for deploying Project-AI services.**

## Usage

```bash
# Install
helm install project-ai helm/project-ai/

# Upgrade
helm upgrade project-ai helm/project-ai/

# Uninstall
helm uninstall project-ai
```
