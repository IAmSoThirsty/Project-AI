# Architecture Documentation

## Overview

System architecture documentation including design decisions, component diagrams, and technical specifications.

## Installation

Documentation is read-only. No installation required.

## Usage

Browse markdown files for architecture information. Use diagrams for visual reference.

## API

Documentation follows ADR (Architecture Decision Records) format.
