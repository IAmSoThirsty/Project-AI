# Agent Configuration

## Overview

Agent configuration directory for Copilot and AI tooling. Contains settings and customizations for intelligent code assistance.

## Installation

No installation required. Configuration files are automatically loaded by the agent runtime.

## Usage

Place agent configuration files in this directory. Files will be automatically detected and applied.

## API

Configuration files use JSON/YAML format. See individual files for specific schema.
