# Codacy Configuration

## Overview

Configuration for Codacy code quality analysis. Contains rules, patterns, and quality gates for automated code review.

## Installation

No installation required. Codacy reads configuration from this directory automatically.

## Usage

Configure code quality rules in `.codacy.yml` and custom patterns in this directory.

## API

Uses Codacy Configuration Schema v2. See https://docs.codacy.com/repositories-configure/codacy-configuration-file/
