```

```

1. **P (Primary Plane)**: Canonical execution path (deterministic function graph)
2. **Sh (Shadow Plane)**: Validation/simulation/containment path (parallel side-effect-bounded graph)
3. **C (Constitutional Core)**: Invariant enforcement & commit authority

