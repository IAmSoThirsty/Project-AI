## 🎯 Path 5: Specific System Deep-Dive

**Goal:** Master one specific system.

Pick a canonical system, then follow this pattern:

1. Read the control node for its domain (e.g., [[Architecture-Map]])
2. Read the canonical system note (e.g., [[PSIA]])
3. Read related publications (linked in frontmatter)
4. Read supporting implementation notes
5. Read runbooks/operational docs
6. Review code in `src/` directory

### Example: Deep-dive on PSIA

1. [[Governance-Map]] or [[Architecture-Map]]
2. [[PSIA]] (canonical)
3. [[OctoReflex-Report]] (related publication)
4. [[Shadow-Execution]] (related system)
5. [[PSIA-Runbook]] (operations)
6. Code: `src/app/core/psia/`
