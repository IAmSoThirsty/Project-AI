2. Scroll through hierarchical sections
3. Follow wiki links to detailed docs

When adding new components:
1. Add entry to relevant index(es)
2. Include all required metadata
3. Add cross-references
4. Update related indexes
5. Validate wiki links

### Quality Standards
All indexes follow:
- **Alphabetical ordering** within sections
- **Consistent formatting** (tables/lists)
- **Complete metadata** (status, audience, tags)
- **Bidirectional links** (no orphans)
