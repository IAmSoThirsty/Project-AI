About these demos

The interactive demos in the library pages are simulation-accurate representations of the actual governance systems. They use the same logic as the production components, running in-browser via Alpine.js. Some demos (OctoReflex live feed, NIRL canvas) are animated simulations of real-time kernel behaviour that cannot run in-browser; they accurately represent the system's operational semantics. The Legion portal runs against the Project-AI agent API.
