Threat Model

What Cerberus defends against.

Blocked · L1
Prompt injection → syscall escape

Adversarial input that attempts to redirect model-produced syscalls outside the governance envelope. OctoReflex intercepts at the syscall surface regardless of model reasoning.

Blocked · L2
Invariant bypass via capability abuse

Actions that request capabilities not granted in the current context. OCEE checks capability bounds before any action proceeds. An ungranted capability cannot be exercised.

Prevented · L4
Guardian exhaustion attack

Attempts to overwhelm the defense layer by saturating guardian capacity. NIRL's exponential spawn model means increased pressure produces increased defensive capacity, not degraded defense.

Blocked · L3
Rate budget overrun

Actions that exceed the rate window defined in constitutional policy. OctoReflex's closed-loop controller tracks cumulative rate across the observation window and denies once budget is exhausted.

Prevented · L5
Unilateral Triumvirate override

Any attempt by a single pillar to unilaterally permit what another has denied. The Triumvirate requires unanimity — not majority, not supermajority. One veto is a denial.
