Formal Proof

Why exploitation cannot finish.

# Let A = attacker resources. Let D(n) = defender processes at depth n.

D(0) = 1 # kernel process
D(n) = 3 × D(n−1) # NIRL spawn law

D(1) = 3 D(2) = 9 D(3) = 27

# To "win," attacker must simultaneously compromise all D(n) processes.
# Each compromise triggers D(n+1) spawn before attacker can act.

∴ Required_A = lim(n→∞) 3ⁿ = ∞

# Saturation cap at n=3 collapses attacker to static isolation.
# No finite attacker can exceed this requirement. QED.
Reflex Properties

Three laws of the immune response.

I

Instantaneous — no deliberation path

The reflex fires at the OS scheduler level. There is no model inference loop, no policy lookup, no quorum consultation. The signal "intrusion detected" triggers guardian spawn as a kernel trap handler — measured in microseconds, not milliseconds.

nirl.on_intrusion → os.spawn_guardian(n=3)
II

Native — operates below the model's privilege ring

"Native" is not a marketing term. NIRL is implemented as a kernel module operating at ring 0. The AI model cannot observe, modify, or disable NIRL from its execution context — which lives in ring 3. The defender is structurally unreachable from the attacker's domain.

ring(nirl) = 0 < ring(model) = 3 → model cannot reach nirl
III

Saturation — finite depth, infinite cost

Guardians spawn to a maximum depth of three generations. At saturation (27 active guardians), NIRL transitions to static isolation mode: the kernel domain is walled, all suspect processes are suspended, and the Cerberus incident log receives a full dump. The attacker is now facing 27 live watchers simultaneously.

depth == 3 → nirl.saturate() → cerberus.incident(dump=True)
Implementation

The reflex in code.

src/cerberus/nirl/reflex.py · NativeImmuneReflex
# NIRL — Native Immune Reflex Layer · Thirsty's Projects LLC
# Ring-0 kernel module interface. Model cannot import or inspect this module.

class NativeImmuneReflex:
  MAX_DEPTH = 3
  SPAWN_FACTOR = 3

  def on_intrusion(self, signal: IntrusionSignal, depth: int = 0) -> None:
    if depth >= self.MAX_DEPTH:
      self.saturate(signal) # static isolation at depth 3
      return

    guardians = [
      Guardian(
        on_compromise=lambda: self.on_intrusion(signal, depth + 1),
        ring=0,
      )
      for _ in range(self.SPAWN_FACTOR)
    ]
    for g in guardians:
      g.deploy()

  def saturate(self, signal: IntrusionSignal) -> None:
    KernelDomain.wall() # suspend all suspect processes
    Cerberus.incident.dump(signal) # full state to incident ledger
    OctoReflex.emergency_gate() # deny all syscalls until cleared
