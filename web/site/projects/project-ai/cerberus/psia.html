Architecture

Three components. One sovereign layer.

Each component has a distinct role. They are not interchangeable and cannot substitute for one another. The architecture is intentionally non-monolithic — a compromise of one component does not grant authority exercised by another.

⚖
Component I · Quorum Engine

Unanimous consent — all three pillars

Elevated capability grants require votes from all three Triumvirate pillars: Galahad (ethics), Cerberus (security), and CodexDeus (language/runtime). All three must vote YES. A single NO vote from any pillar blocks the grant — there is no majority, no tie-breaking, no override.

quorum.vote(pillar, capability) → YES | NO
📒
Component II · PSIA Ledger

Append-only record of every arbitration

Every quorum vote, every capability grant, every DENY decision, and every expiry event is recorded in the PSIA Ledger. The ledger is append-only and cryptographically chained — each entry seals the previous. Nothing can be removed. Audit is permanent by design.

ledger.append(event, seal=ed25519, chain=sha256)
🔑
Component III · Capability Authority

Time-bounded permissions — provisional by default

Following a successful quorum vote, the Capability Authority issues a time-bounded capability token. There are no permanent grants. Every capability expires. Renewal requires a new quorum vote. A revoked capability cannot be re-activated — only newly granted by fresh quorum.

capability.grant(action, ttl=3600, revocable=True)

Implementation

PSIA in code.

src/psia/gate/quorum_engine.py · QuorumEngine.arbitrate
# PSIA — Provisional Sovereign Intelligence Arbitration · Thirsty's Projects LLC

class QuorumEngine:
  def arbitrate(self, request: CapabilityRequest) -> ArbitrationResult:
    votes: dict[Pillar, Vote] = {}

    for pillar in (Galahad, Cerberus, CodexDeus):
      votes[pillar] = pillar.vote(request)
      if votes[pillar] == Vote.NO:
        # One NO immediately closes the vote — no point continuing
        break

    unanimous = all(v == Vote.YES for v in votes.values())
    result = ArbitrationResult(
      granted=unanimous,
      votes=votes,
      capability=self._cap_authority.issue(request) if unanimous else None,
    )
    self._ledger.append(result) # seal regardless of outcome
    return result
