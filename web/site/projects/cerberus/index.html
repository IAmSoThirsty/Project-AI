Cerberus coordinates three guardian types in parallel — Pattern (regex threat signatures), Heuristic (behavioral analysis), and Statistical (entropy/anomaly detection). Architecture

Three guardian types. One coordinator hub.

Guardian Type 01

Pattern Guardian

Regex threat signature library. Covers prompt injection, jailbreak, SQL injection, command injection, data exfiltration, and AI-specific attack vectors.

Guardian Type 02

Heuristic Guardian

Behavioral analysis. Detects untrusted actors, external origins requesting write access, and anomalous action sequences that pattern matching alone would miss.

Guardian Type 03

Statistical Guardian

Entropy and anomaly detection. Identifies statistically unusual input patterns that indicate adversarial probing even when individual requests appear benign.

Coordinator

CerberusHub

Coordinates all three guardians simultaneously. Exponential defense: bypass attempt → 3 new guardian spawn → max 27 → auto-shutdown at ceiling.
