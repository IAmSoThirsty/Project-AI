## Usage Examples

### Basic Projection

```python
from atlas.simulation.monte_carlo_engine import get_monte_carlo_engine, WorldState

engine = get_monte_carlo_engine(seed="0xA17F01")
initial_state = WorldState(timestamp=datetime.now(), timestep=0, ...)
engine.set_initial_state(initial_state)
states = engine.run(n_steps=100)
```

### Multi-Seed Analysis

```python
from atlas.simulation.timeline_divergence import get_timeline_divergence_engine

engine = get_timeline_divergence_engine()
tensor = engine.project_all_timelines(initial_state, steps_per_year=12)
print(f"Projected {tensor.get_statistics()['total_points']} points")
```

### Query Validation

```python
from atlas.safeguards.epistemic_safeguards import get_epistemic_safeguards

safeguards = get_epistemic_safeguards()
validation = safeguards.framing_guards.validate_query("What should we do?")
print(f"Allowed: {validation.is_allowed}")  # False - normative query
```

### Decision Logging

```python
from atlas.safeguards.epistemic_safeguards import Decision, DecisionBasis

decision = Decision(
    decision_id="dec_001",
    decision_maker="Alice (Triumvirate)",
    description="Approve policy",
    atlas_consulted=True,
    basis=DecisionBasis.INFORMED_AND_VALIDATED,
    reasoning_beyond_atlas=["Historical precedent", "Expert consultation"]
)
safeguards.gravity_mitigation.log_decision(decision)
```
