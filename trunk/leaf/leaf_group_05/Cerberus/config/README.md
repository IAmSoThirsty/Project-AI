# Cerberus Guard Bot Configuration

This directory contains configuration files for the Cerberus Guard Bot.

## Files

- `default.yaml` - Default configuration values
- `production.yaml` - Production environment settings (to be added)
- `development.yaml` - Development environment settings (to be added)

## Usage

Configuration is loaded based on the `CERBERUS_ENV` environment variable.
Default is `development`.
