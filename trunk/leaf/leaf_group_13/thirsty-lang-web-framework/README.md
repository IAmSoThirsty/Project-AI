# Thirsty-lang Web Framework 💧

Build web applications with Thirsty-lang's water-themed syntax.

## Features

- Router system
- Middleware support
- Template rendering
- Session management
- Example blog application

## Quick Start

```thirsty
// Simple server
glass createServer(port) {
shield webProtection {
    drink server = WebServer(port)
    server.listen()
  }
}
```
