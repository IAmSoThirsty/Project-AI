# Exercise 01: Hello Hydration

**Difficulty**: ⭐ Beginner  
**Topics**: Output, Strings  
**Time**: 5 minutes

## 📝 Description

Write a program that outputs "Stay hydrated!" to the console.

## 🎯 Learning Objectives

- Use the `pour` keyword for output
- Understand string literals
- Write your first Thirsty-lang program

## 📋 Requirements

Your program should:

1. Output the exact text: "Stay hydrated!"
2. Use the `pour` keyword

## ✅ Expected Output

```
Stay hydrated!
```

## 💡 Hints

- Remember that `pour` is used for console output
- Strings are enclosed in double quotes
- End each statement (no semicolons needed!)

## 🚀 Bonus Challenge

Modify your program to also output:

- "Drink 8 glasses a day!"
- "Water is essential! 💧"

## 📖 Relevant Concepts

- `pour` - Output keyword
- String literals
