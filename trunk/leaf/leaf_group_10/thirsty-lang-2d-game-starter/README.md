# Thirsty-lang 2D Game Starter 💧🎮

Ready-to-use 2D game template with sprites, animations, physics, and input handling.

## Included

- Sprite rendering system
- Animation controller
- 2D physics with collision
- Particle effects
- Sound manager
- Level editor
- Example: Space shooter game

## Quick Start

```thirsty
import { Game } from "game engine"

glass main() {
  drink game = Game(800, 600, "My 2D Game")
  game.start()
}
```

## Features

- Entity-component system
- Tilemap support
- Camera system
- Input handling (keyboard/mouse/gamepad)
- Asset loading
- Scene management
