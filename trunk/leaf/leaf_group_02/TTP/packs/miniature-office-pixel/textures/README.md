# Miniature Office Pixel Textures

This pack contains pixel-perfect textures for the Miniature Office scene.

- `office_desk_albedo.png` (64x64) - Desk base color
- `office_desk_normal.png` (64x64) - Desk normal map
- `office_chair_albedo.png` (32x32) - Chair base color
- `office_chair_normal.png` (32x32) - Chair normal map

## Usage

These textures are designed for pixel art games with a retro aesthetic. They use a limited color palette and crisp pixel edges.

### Technical Specs

- Format: PNG
- Color Space: sRGB
- Alpha: Premultiplied where applicable
- Tiling: Most textures are designed to tile seamlessly

### Recommended Settings

- **Filter Mode**: Point (No Filter)
- **Mip Maps**: Disabled
- **Compression**: None or lossless

## Color Palette

Primary colors used:
- Wood: #8B6F47, #6B5638
- Metal: #A8A8A8, #707070
- Screen: #00FF00 (emission)
- Accents: #FF6B6B, #4ECDC4
