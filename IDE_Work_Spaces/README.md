# IDE Work Spaces

This folder holds focused workspace wiring for Project-AI.

The root `Project-AI.code-workspace` remains the broad workspace that records
the wider local ecosystem. `project-ai-control-plane.code-workspace` is the
compact Heart Restore workspace for the live control plane.

Open the compact workspace when validating:

- `src/app/core/`
- `src/app/agents/`
- `src/app/governance/`
- `governance/`
- `cognition/`
- `canonical/`
- `config/`
- `.github/workflows/`
- `docs/operations/`

Run `.\FIX_WORKSTATION.ps1` from the repo root to verify the paths before any
structural cleanup.
