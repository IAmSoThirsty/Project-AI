# Staging package surfaces

Everything under this directory is retained as staging, legacy, or reference
material. “Production-ready” language inside a staging artifact describes that
artifact's implementation notes only; it is not approval to deploy the
Project-AI runtime. Current release status is governed by
`docs/deployment/PRE_DEPLOYMENT_CHECKLIST.md` and
`docs/operations/cab/PROJECT_AI_V0.0.3_SUCCESSOR_CAB_REVIEW_PACK.md`.
Deployment approval remains fail-closed until those successor gates pass.
