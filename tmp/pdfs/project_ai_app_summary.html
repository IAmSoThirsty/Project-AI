What It Is

Project-AI is a governance-first AI app/platform that combines a desktop shell, a Python trust-root runtime, and a FastAPI governance host. It routes work through TARL rules, a CognitionKernel, and auditable control paths.

What It Does

  • Evaluates intents through /intent and returns allow / deny / degrade verdicts.
  • Shows health, TARL rules, and audit records in the desktop UI.
  • Boots CognitionKernel with identity, memory, governance, reflection, and CouncilHub routing.
  • Loads defense and oversight agents, including SafetyGuard, constitutional guardrail, TARL protector, validator, explainability, and red-team roles.
  • Exposes workspace browsing, repo status, local terminal sessions, and guarded browser opening from Electron.
  • Offers Electron/React desktop and a PyQt6 "Leather Book" UI.
  • Adds monitoring, Temporal, and microservices through Docker Compose.

How It Works

Shells Electron/React desktop runs from desktop/. The repo launcher can start that shell or switch to the PyQt6 LeatherBookInterface.
API start.ps1 starts uvicorn psia.server.governance_server:app on 127.0.0.1:8001; the desktop client calls /health, /tarl, /audit, and /intent.
Trust Root src/app/main.py initializes the tier registry, CognitionKernel, IdentityIntegratedIntelligenceEngine, MemoryEngine, governance Triumvirate, ReflectionCycle, and CouncilHub.
Security + Data Security systems include GlobalWatchTower, agent guards, ASL-3 enforcement, IP blocking, and honeypot detection. Runtime state spans data/, logs/, config/, and governance/; docker-compose.yml adds monitoring, Temporal, and extra services.

How To Run

  1. Install Python 3.11+ and Node.js. Docker is optional.
  2. From repo root: pip install -r requirements.txt
  3. Launch the desktop path on Windows: .\start.ps1
  4. Optional full PyQt UI: .\start.ps1 -Full

Repo note: start.ps1 auto-runs npm install in desktop/ if needed.
