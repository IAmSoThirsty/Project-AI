# Project-AI Datasets

This directory is the designated location for all datasets used by Project-AI.

## Structure

- `raw/`: Unprocessed, original data.
- `processed/`: Cleaned and transformed data ready for model consumption.
- `external/`: Data from third-party sources.

## Governance

All data stored here must adhere to Project-AI's governance protocols (PAGL). Ensure no PII (Personally Identifiable Information) is stored without proper encryption and consent.
