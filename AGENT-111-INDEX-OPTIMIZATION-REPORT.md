### Workflow 2: Index Link Validation (CI/CD)

**Trigger:** Pre-commit hook or CI pipeline  
**Action:** Validate all index links and update broken link report  
**Tool:** Custom Python script + GitHub Actions

**Validation Script:**
```python
# validate_indexes.py
import re
from pathlib import Path

def validate_index_links(index_file):
    """Validate all wiki links in an index file."""
    with open(index_file) as f:
        content = f.read()
    
    # Extract all wiki links
    links = re.findall(r'\[\[(.*?)\]\]', content)
    
    broken_links = []
    for link in links:
        target = link.split('|')[0]  # Handle aliased links
        if not Path(target).exists():
            broken_links.append(link)
    
    return broken_links

# Run validation on all indexes
indexes = Path('.').rglob('*INDEX*.md')
for index in indexes:
    broken = validate_index_links(index)
    if broken:
        print(f"❌ {index}: {len(broken)} broken links")
    else:
        print(f"✅ {index}: All links valid")
```

  schedule:
    - cron: '0 0 * * 0'  # Weekly on Sunday
jobs:
  detect-orphans:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Run Dataview orphan query
        run: |
          # Execute Query 6 and save results
          obsidian-cli query "orphan-detection.dataview" > orphans.txt
      - name: Create GitHub Issue if orphans found
        if: file_size(orphans.txt) > 0
        uses: actions/github-script@v6
        with:
          script: |
            github.rest.issues.create({
              owner: context.repo.owner,
              repo: context.repo.repo,
              title: 'Orphaned Documents Detected',
              body: file('orphans.txt'),
              labels: ['documentation', 'maintenance']
            })
