## Custom Hotkeys

### How to Customize

**Steps:**

1. **Open Settings**
   - `Ctrl+,` (Windows) or `Cmd+,` (Mac)

2. **Navigate to Hotkeys**
   - Settings → Hotkeys

3. **Search for Command**
   - Type command name in search box
   - Example: "Templater: Insert template"

4. **Assign Shortcut**
   - Click in hotkey field
   - Press desired key combination
   - Obsidian warns if conflict exists

5. **Save**
   - Automatic save

### Recommended Custom Hotkeys

| Command | Suggested Shortcut | Why |
|---------|-------------------|-----|
| **Insert Template** | `Alt+E` | Quick template access |
| **Toggle Left Sidebar** | `Ctrl+Alt+L` | Show/hide file explorer |
| **Toggle Right Sidebar** | `Ctrl+Alt+R` | Show/hide metadata |
| **Open Graph View** | `Ctrl+G` | Visualize connections |

| **Create Daily Note** | `Ctrl+Alt+D` | Quick daily note |
| **Open Random Note** | `Ctrl+Alt+N` | Serendipitous discovery |

## Productivity Tips

### Tip 1: Master the Command Palette

**Why:** Access ANY command without memorizing shortcuts

**How:**
```
1. Press Ctrl+P (Cmd+P on Mac)
2. Type what you want to do: "insert", "graph", "export"
3. Select from filtered list
4. Press Enter
```

**Examples:**
```
Want to export? Type "export pdf"
Need to validate? Type "validate"
Insert template? Type "template"
```

**Pro Tip:** Command palette learns your usage. Frequent commands appear first.

### Tip 2: Use Quick Switcher Fuzzy Search

**Example:**

```
Filename: ARCHITECTURE_DECISION_RECORD_023.md

Type: "adr23"
Result: Matches! (architecture-decision-record-023)

Type: "archdes"
Result: Matches! (architecture-design)

Type: "023"
Result: Matches! (any file with 023)
```

**Pro Tip:** Use capital letters as shortcuts - "ADR" finds Architecture Decision Records

### Tip 3: Multi-Cursor Editing

**Use Case:** Update multiple similar lines

**How:**

```
1. Select first instance (Ctrl+D)
2. Press Ctrl+D again to select next occurrence
3. Keep pressing Ctrl+D for more
4. Edit all simultaneously

Example:
tags: [architecture]    → tags: [architecture, security]
tags: [development]     → tags: [development, security]
tags: [operations]      → tags: [operations, security]

Just press Ctrl+D until all "tags:" selected, then edit once!
```

### Tip 4: Link While Typing

**Fast Link Creation:**

```
Type: [[
→ Autocomplete shows file list
Type: arch
→ Filters to architecture files
Press: Arrow keys + Enter
→ Link inserted!

No mouse needed!
```

### Tip 5: Navigate History Like a Browser

**Shortcuts:**

```
Ctrl+Alt+← (Cmd+Opt+← Mac) → Back
Ctrl+Alt+→ (Cmd+Opt+→ Mac) → Forward

Works exactly like browser back/forward!
```

### Tip 6: Workspace Shortcuts

**Setup:**

```
Save workspace layouts:
1. Arrange panes how you like
2. Ctrl+P → "Save workspace"
3. Name it (e.g., "Documentation Mode")

Load workspace:
1. Ctrl+P → "Load workspace"
2. Select saved workspace
3. Instant layout restore!
```

**Use Cases:**
- Writing Mode: Single pane, distraction-free
- Research Mode: Split panes with graph
- Review Mode: Side-by-side comparison

## Platform Differences

### Windows vs macOS

**Modifier Keys:**

| Windows | macOS | Purpose |
|---------|-------|---------|
| `Ctrl` | `Cmd` | Primary modifier |
| `Alt` | `Opt` | Secondary modifier |
| `Windows` key | N/A | OS-level shortcuts |

**Common Translations:**

```
Windows          macOS
─────────────────────────────────
Ctrl+C           Cmd+C   (Copy)
Ctrl+V           Cmd+V   (Paste)
Ctrl+Z           Cmd+Z   (Undo)
Ctrl+O           Cmd+O   (Open)
Alt+Click        Opt+Click (Multi-cursor)
```

### Linux Specifics

**Generally follows Windows conventions:**
- `Ctrl` is primary modifier
- `Alt` is secondary modifier
- Some window manager shortcuts may conflict (e.g., `Alt+F4`)

**Workarounds:**
- Customize conflicting hotkeys in Obsidian
- Modify window manager shortcuts
- Use alternative key combinations
