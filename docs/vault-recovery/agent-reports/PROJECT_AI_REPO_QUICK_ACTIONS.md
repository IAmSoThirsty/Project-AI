# Project-AI Quick Actions

Use these prompts directly in your local Obsidian Copilot chat.

## Navigation

- "Find where governance mutation flow is implemented and summarize it."
- "List all files related to constitutional AI systems and what each does."
- "Show me where Desktop and Web API stacks diverge."

## Implementation Support

- "Given this task, identify exact files to edit and testing scope."
- "Create a safe step-by-step patch plan with rollback notes."
- "Explain side effects before modifying shared pipeline code."

## Verification

- "What tests should I run for this change and why?"
- "Summarize probable hidden-test risks for this patch."
- "Check if this change violates governance/profile constraints."

## Coordination (Human + AI)

- "Convert this request into owner/task/dependency/done checklist."
- "Draft an AI-to-human handoff update with evidence and next step."
- "Draft a reviewer note for PR discussion with risks and mitigations."

## Troubleshooting Local AI

- If model errors appear, verify:
  - LM Studio server is running
  - Endpoint is `http://127.0.0.1:1234/v1`
  - Obsidian model is `obsidian-agent|lm-studio`
  - CORS is enabled on LM Studio server
