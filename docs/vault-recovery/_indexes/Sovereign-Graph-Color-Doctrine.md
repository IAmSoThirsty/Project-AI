# Sovereign Graph Color Doctrine

Central Index: [[Sovereign Vault Index]]

The vault graph uses a three-color doctrine aligned to the Project-AI triumvirate.

## Cardinal Red - Cerberus

Security, defense, containment, red-team, threat response, vulnerability tracking, and safety guardrails.

Primary color: `#C41E3A`

## Blue - Galahad

Cognitive, moral, ethical, philosophical, constitutional, persona, and user-facing intelligence systems.

Primary color: `#2F6FFF`

## Gold - Codex Deus Maximus

Truth, knowledge, arbitration, source-of-record material, central indexes, MOCs, and navigation.

Primary color: `#D4AF37`

## Edge Glow Concept

Obsidian's built-in graph settings can color nodes by query group, but they do not expose per-edge color, glow, or split-gradient controls. A red-to-blue edge, half Cardinal Red and half Blue, would require a custom graph plugin or an exported custom graph renderer.

The current stock-Obsidian approximation is:

- color the linked nodes by doctrine
- keep arrows enabled
- keep link thickness slightly raised
- preserve the central hub as the dominant gold knowledge node
