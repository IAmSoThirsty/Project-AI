# Architecture Documentation

This folder contains detailed technical diagrams, system architecture documents, and unified architecture overviews.

## What Belongs Here

- **System architecture documents** - High-level system design and structure
- **Technical diagrams** - Architecture diagrams, data flow diagrams, component diagrams
- **Integration architecture** - How different systems and components integrate
- **Infrastructure architecture** - Deployment and infrastructure design
- **Platform architecture** - Multi-platform architecture documentation

## Key Documents

- `ARCHITECTURE_OVERVIEW.md` - Complete system architecture overview
- `AGENT_MODEL.md` - PACE agent architecture and coordination model
- `../templates/AGENT_BUILD_TEMPLATE.md` - Practical template for defining and building new agents
- `PROJECT_AI_KERNEL_ARCHITECTURE.md` - Kernel architecture details
- `KERNEL_MODULARIZATION_SUMMARY.md` - Modularization approach
- `HYDRA_50_ARCHITECTURE.md` - High-availability architecture
- `TEMPORAL_INTEGRATION_ARCHITECTURE.md` - Temporal workflow architecture

## Guidelines

- **Audience**: Technical architects, senior engineers, technical decision-makers
- **Tone**: Technical, precise, comprehensive
- **Content**: System design, component interactions, architectural patterns
- **Include**: Diagrams, ASCII art, technical specifications
