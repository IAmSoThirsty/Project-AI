# TSCG

## Definition
Symbolic grammar for encoding governance decisions.

## Purpose
Compress complex decision logic into structured form.

## Core Elements
- COG (Cognition)
- SHD (Shadow)
- INV (Invariants)
- QRM (Quorum)
- COM (Commit)
- ANC (Anchor)

## Property
Deterministic and compact representation of decision flow.
