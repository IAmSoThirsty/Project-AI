Privacy Policy

How we protect your privacy while using this website

Last Updated: February 8, 2024

Overview

The Epstein Files Hub is committed to protecting your privacy. This policy explains how we handle information when you use our website.

TL;DR: We collect minimal data. Search is private. No tracking. No ads. Open source code.

Information We Do NOT Collect

We prioritize privacy and do NOT collect:

  • Search Queries: Your searches are performed entirely in your browser - we never see them
  • Personal Information: No names, emails, phone numbers, or addresses
  • User Accounts: No login required, no user profiles
  • Browsing History: We don't track which pages you visit
  • Cookies for Tracking: No advertising or tracking cookies
  • Device Fingerprinting: No attempts to uniquely identify your device

Information We DO Collect

We collect minimal information necessary for site operation:

Basic Server Logs (GitHub Pages)

GitHub Pages automatically logs:

  • IP address (anonymized)
  • Browser type and version
  • Date and time of access
  • Pages requested

This data is collected by GitHub, not us, and is used solely for security and performance monitoring. See GitHub's Privacy Policy.

Cloudflare CDN

Our CDN provider (Cloudflare) collects:

  • IP address (for DDoS protection)
  • Request headers
  • Performance metrics

See Cloudflare's Privacy Policy.

GitHub Submissions

If you contribute via GitHub:

  • Your GitHub username is public
  • Contribution metadata (time, commit message)
  • Any information you choose to include

This is controlled by GitHub's platform. See GitHub's Privacy Policy.

How We Use Information

The minimal data collected is used only for:

  • Security: Detecting and preventing attacks
  • Performance: Optimizing site speed and reliability
  • Troubleshooting: Fixing technical issues
  • Compliance: Meeting legal obligations

We do NOT:

  • Sell or share data with third parties
  • Use data for advertising or marketing
  • Track individual users across sessions
  • Build user profiles or behavior models

Client-Side Search Privacy

Our search functionality is designed for maximum privacy:

🔒

Local Processing

All searches run in your browser using JavaScript. The search index is downloaded once, then all queries are processed locally.

🚫

No Server Queries

Your search terms never leave your device. There are no API calls to our servers for search queries.

📴

Offline Capable

Once the page and search index are loaded, you can search even without an internet connection.

👁️

No Tracking

We don't log, store, or analyze what you search for. Your research is completely private.

Cookies

We use minimal cookies:

  • Session Cookies: For basic site functionality (if needed)
  • Cloudflare Cookies: For security and performance (minimal)

We do NOT use:

  • Advertising cookies
  • Third-party tracking cookies
  • Analytics cookies (beyond GitHub Pages basic stats)
  • Social media cookies

Third-Party Services

We use the following third-party services:

GitHub Pages (Hosting)

Our website is hosted on GitHub Pages, which provides basic server logs. GitHub Privacy Policy

Cloudflare (CDN & Security)

Cloudflare provides content delivery and DDoS protection. Cloudflare Privacy Policy

GitHub (Contributions)

If you contribute via GitHub, their privacy policy applies. GitHub Privacy Policy

Data Retention

Minimal data is retained:

  • Server Logs: Retained by GitHub/Cloudflare per their policies (typically 30-90 days)
  • Contributions: Permanently stored in Git history (open source)
  • Search Data: Not collected or retained

Your Rights

You have the right to:

  • Access: Request any data we may have about you (likely none)
  • Deletion: Request deletion of data (where technically possible)
  • Correction: Request correction of inaccurate data
  • Objection: Object to data processing (though we do minimal processing)
  • Portability: Receive data in a portable format

To exercise these rights, open an issue on GitHub.

Children's Privacy

This website is not directed at children under 18. The content contains information about serious crimes and is intended for adult researchers, journalists, and the general public. We do not knowingly collect data from children.

International Users

This website is accessible globally. If you access from outside the United States:

  • Your data may be transmitted to US-based servers (GitHub, Cloudflare)
  • We comply with applicable international privacy laws (GDPR, CCPA, etc.)
  • You have the same privacy rights as US users

Security

We employ security measures including:

  • HTTPS/SSL encryption for all connections
  • Cloudflare DDoS protection
  • Regular security audits via GitHub Actions
  • Open source code (community security review)
  • Minimal data collection (less data = less risk)

Changes to Privacy Policy

We may update this policy occasionally. Changes will be:

  • Posted on this page with updated date
  • Noted in our changelog
  • Tracked in Git history (full transparency)

Continued use after changes constitutes acceptance.

  • GitHub Discussions
