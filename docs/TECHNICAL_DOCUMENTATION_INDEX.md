## Key Terminology

**Core Concepts:**

- **FourLaws:** Immutable ethical framework (Asimov-inspired)
- **AIPersona:** Self-aware AI with personality and mood
- **Memory Expansion:** Persistent knowledge base (6 categories)
- **Learning Request:** Human-in-the-loop approval workflow
- **Black Vault:** SHA-256 fingerprinting of denied content
- **Command Override:** Master password emergency control
- **Plugin System:** Extensibility framework
- **Oversight Agent:** Pre-execution safety validation
- **Planner Agent:** Task decomposition
- **Validator Agent:** Input/output sanitization
- **Explainability Agent:** Decision transparency
- **Leather Book:** Tron-inspired desktop UI
- **SLSA Level 3:** Supply chain security standard
- **PAGL:** Project-AI Governance License
