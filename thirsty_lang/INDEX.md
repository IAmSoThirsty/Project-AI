## 🔧 Quick Commands

```bash
# Installation
pip install z3-solver

# Run demo
python demo_enhanced.py

# Analyze a file
python -m src.shadow_enhanced your_file.shadow

# Security scan (fast)
python -m src.shadow_enhanced file.shadow --performance --no-tests

# Generate tests
python -m src.shadow_enhanced file.shadow --coverage 95.0

# Full analysis with reports
python -m src.shadow_enhanced file.shadow --output ./reports --verbose
```
