# Thirst of Gods Manifesto

This stack exists because code should feel like a living current, not dead syntax.
Thirsty-Lang speaks in springs, wells, reservoirs, floods, and condensation because the language is supposed to carry intention in its grammar.

The soul rules:
- every command should feel poured, not bolted on
- metaphor should increase clarity, not obscure it
- one thirsty command should feel like commanding a living system
